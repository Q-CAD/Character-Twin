# Character-Twin
Codes to simulate standard characterization protocols to assess quality and properties of thin-films in- an ex-situ; 
Characterizations such as reflection high-energy electron diffraction (RHEED), convergent beam electron diffraction (CBED), selected area electron diffraction (SAED), electron energy loss spectra (EELS) and x-ray diffraction (XRD) will be included here.
